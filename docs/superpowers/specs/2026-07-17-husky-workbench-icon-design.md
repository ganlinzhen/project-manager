# 哈士奇工作台应用图标设计

## 目标

为“工作管理器”桌面应用设计一枚有真实犬种特征、适合 macOS Dock 与应用内品牌位使用的图标。它应延续产品“沉着、清晰、可信”的气质，不采用过度卡通或复杂的营销插画风格。

## 最终方向

采用“沉着的半写实哈士奇”。构图为居中的哈士奇上半身：灰白相间的毛发、冰蓝色双眼、细框眼镜；前方是一台打开的银色 MacBook，前爪自然落在电脑后侧。背景采用现有设计系统的深氧化青色，笔记本与眼镜保持低对比中性色，避免与背景争夺焦点。

## 视觉约束

- 图标为圆角方形，适合 512×512 源图，并能在 16px 至 128px 尺寸保持主体可辨。
- 哈士奇的脸、眼镜和笔记本是三个最重要的识别层级；背景不加入桌面、房间、文字、徽标或额外物件。
- 半写实、精致数字插画风格，毛发有自然质感但不使用细碎到影响缩小识别的纹理。
- 表情保持专注、温和，不露出夸张笑容，不使用霓虹色、玻璃拟态或深色黑客终端视觉。
- 不在图中渲染 Apple 标志、应用名称、水印或文字。

## 交付物

- 一张 512×512 的 PNG 图标预览与项目内最终资产。
- 视觉检查：中心构图、边缘留白、冰蓝眼睛和眼镜在小尺寸下可见，且无文字、水印或第三方品牌标志。
